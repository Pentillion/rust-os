# Build an OS in Rust - Learning Project

This repository follows the ["Writing an OS in Rust"](https://os.phil-opp.com/) series by Philipp Oppermann.

Goals:

- Learn low-level systems programming
- Understand bootloaders, paging, interrupts
- Explore Rust in no_std environments

Status:

- [x] Bare metal boot
- [ ] VGA text buffer
- [ ] Interrupts
- [ ] Memory management
